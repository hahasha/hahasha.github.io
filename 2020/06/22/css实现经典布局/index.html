三栏布局

三栏布局的特点是：两边定宽、中间自适应。
这里主要介绍三栏布局的两种实现方式，一种是圣杯布局，来源于文章In Search of the Holy Grail，另一种是双飞翼布局，来源于淘宝UED。

1. 圣杯布局

特点：

  1. header 和 footer 宽度占满全屏；
  2. 中间部分是一个三栏布局，两侧定宽，中间栏自适应；
  3. 中间栏在DOM结构上优先，以便先行渲染；
  4. 中间部分的高度是三栏中最高部分的高度。
